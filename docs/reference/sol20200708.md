# Reference strike() solution

This was produced with the package as it existed on 2020-jul-8, prior to
the publication of Kelley et al. (2021). It is used in testing, to
ensure that the package does not inadvertently change in its
predictions.

## References

Kelley, Dan E., James P. Vlasic, and Sean W. Brillant. "Assessing the
Lethality of Ship Strikes on Whales Using Simple Biophysical Models."
Marine Mammal Science, 37(1), 2021, mms.12745.
[doi:10.1111/mms.12745](https://doi.org/10.1111/mms.12745) .
